# 🌟 English App Recommender
AI 기반 설문으로 **60개 영어 학습 앱 중 나에게 가장 맞는 TOP 3 앱**을 추천해주는 맞춤형 웹 서비스

---

## 🚀 Demo Links
- **Cloudflare Pages:** https://english-app-recommender.pages.dev  
- **GitHub Pages:** https://soumincor.github.io/english-app-recommender/

---

## 🎯 서비스 소개
수많은 영업 앱 중에 뭘 써야 할지 고민하는 사람들을 위한 추천 서비스  
10개 질문을 통해 사용자의 목적/수준/예산/선호도를 분석한 뒤,  
AI 매칭 로직으로 60개 앱의 적합도를 계산하여 추천

---

## ✨ 주요 기능

### ✔ 10문항 설문 기반 추천
- 학습 목적  
- 영어 레벨  
- 시험 대비 여부  
- 예산  
- 공부 시간  
- 학습 스타일 등

### ✔ 60개 앱 태그 매칭 알고리즘
각 앱은 다음과 같은 태그로 분류되어 비교됩
- #회화 #비즈니스 #시험  
- #초보 #중급 #고급  
- #무료 #유료  
- #튜터링 #게임형 #패턴학습 #영상학습  
- #AI보조 #키즈학습  

사용자가 선택한 조건과 앱의 태그를 매칭하여 적합도 스코어를 계산

### ✔ 결과 페이지 기능
- TOP 3 앱 상세 정보  
- 추가 추천 앱 목록  
- iOS / Android / 웹사이트 링크 제공  
- 선택 이유·학습 팁 제공
