# 영어 앱 추천 (Light Version)

AI가 60개 영어 학습 앱 중 지금 나에게 가장 잘 맞는 TOP 3 앱을 추천해주는 간단한 웹페이지입니다.

## 🌐 데모
- Cloudflare Pages: https://english-app-recommender.pages.dev
- GitHub Pages: https://soumincor.github.io/english-app-recommender/

## ✨ 기능 요약
- 10가지 질문에 답하면 나에게 맞는 영어 학습 앱 추천
- 60개 앱 데이터 기반 적합도 계산
- TOP 3 앱 + 추가 후보 앱 제공
- iOS/Android/공식 사이트 바로가기 버튼 제공
- 단일 페이지로 간단하고 빠른 사용성
